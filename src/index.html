SPACE SHOOTER

Space Shooter is a Space Invaders inspired game that allows the player to traverse through hundreds (technically an infinite number) of levels that progress in difficulty and speed. Within each level, there are four types of ships ranked easy, medium, hard, and boss. Each ship level contains its own artwork, health, and missile speed. Once the player successfully destroys the boss at that particular level, the game level will increase, increasing the stats of each enemy ship, making the game fun yet progressively challenging.

BUDGET TRACKER

Budget tracker is an idea I discovered from a Udemy instructor. One of the final projects of the course was to create a budget application that could keep track of a list of incomes and expenses that made budgeting simple, easy, and enjoyable. Budget tracker allows the user to keep a list of incomes and expenses with their corresponding description and prices to help the user remember every detail when budgeting. The application automatically updates the total income made along with the amount the user has spent. In addition, a total budget is calculated that substracts the total income from the total expenses to show the user how much money is left over. One final feature allows users the ability to easily remove items within the list.

SNAKE

This is the first SFML game that I made to test out the capability of the Simple Fast Multimedia Library. The game has a working menu that allows the option of playing or exiting the game. Within the game, the snake can collide with the walls or itself, causing the game to restart. There is also an active score tracker to keep track of your highest score. To make the game more exciting, I added sound effects whenever the snake eats an apple or collides with the wall/itself.
